### Semplice base per una esercitazione sulle classi orientate ad oggetti
---
### Contiene:
- Classi di IO
- Classe per la lettura delle risorse
- Dipendenza con la libreria esterna Gson

Il resto dell'implementazione è compito dello studente, che ha carta bianca fintanto che segue le [funzionalità](https://docs.google.com/document/d/1tzVQwvd3Fe3RkVEtRnvO1kXkd_irqYxTxdyh_EUutNs/edit?usp=sharing).

Nelle risorse è già presente un json con tutti i pokémon della prima generazione e nella cartella principale un file batch per avviare il progetto.
